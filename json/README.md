

* 엔진에서는 input.json을 제공해주어야 한다.
* UI에서는 input.json을받아서,
  * diffcode 각 라인을 클릭했을때 영향받는 코드들을 나열해주는 이벤트를 보여야 한다. (빠진코드는 빨간색, 더해진코드는 초록색 음영으로 표시)
  * 영향받는 코드들은 나열해준 뒤에, "from"프로퍼티를 참조하여 diffcode에서 "클릭한 라인"이 "from"에 나타나있는 코드만 하이라이팅 해준다. (음영을 노란색으로 칠한다던가)
  * 빠진코드를 클릭했을때는 일단 신경쓰지 않는다. 이벤트를 추가하지 않아도 된다.

* 일단 1차로 위의 요구사항을 만족하는 UI를 만들고, 코드블록 사이의 그래프나 브랜치 그래프는 2차 요구사항을 정의하고 진행한다.
